GET /disasters/details

Récupère un package complet de statistiques pour un pays spécifique via son identifiant unique (**FID**). Cette route agrège les données de mortalité, les impacts économiques et les tendances historiques.

Query Parameters

fid number (Required)
info Usage UI

Principalement utilisé pour alimenter le Drawer latéral lors d'un clic sur la carte OpenLayers.
